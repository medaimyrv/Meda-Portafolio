# Portafolio - Medaimy Rivera Valiente

Portafolio profesional de Medaimy Rivera Valiente, Ingeniera Informatica graduada de la UCI, especializada en Inteligencia Artificial y desarrollo web moderno.

**[Ver en vivo](https://meda-portafolio.vercel.app/)**

## Stack Tecnologico

### Portafolio construido con
- **Next.js 14** - Framework React con App Router
- **TypeScript** - Tipado estatico
- **Tailwind CSS** - Estilos utilitarios
- **Framer Motion** - Animaciones
- **React Icons** - Iconografia

### Desplegado en
- **Vercel** - Hosting y CI/CD automatico
- **GitHub** - Control de versiones
